💭 Заметка: Grid vs Flexbox

🔹 Grid — идеален для двумерных структур (шапка, сайдбар, контент, футер), обеспечивает полный контроль над строками и колонками одновременно.
🔸 Flex — лучше для одномерных раскладок: навигационные панели, выравнивание элементов в строку/колонку.
🧪 В данной лабораторной работе наглядно показаны основные приёмы Grid, которые можно миксовать с Flex в реальных проектах.
