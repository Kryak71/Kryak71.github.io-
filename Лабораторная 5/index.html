📐 1. Блочная модель в CSS

Каждый HTML-элемент — это прямоугольный контейнер. Блочная модель описывает, как рассчитывается его итоговый размер:

  • content — содержимое
  • padding — внутренний отступ
  • border — рамка
  • margin — внешний отступ

🎯 Наглядный пример

🔥 Блок 🔥

Разбор параметров:

  • width: 220px; height: 100px; — размер области содержимого
  • padding: 24px; — добавляет отступы внутри со всех сторон
  • border: 6px solid #FFD966; — рамка толщиной 6px
  • margin: 36px; — внешний отступ от соседей
  • Итоговая ширина блока = 220 + 24*2 + 6*2 = 280px (без учёта margin)

💡 box-sizing: border-box — меняет поведение: width/height включают padding и border.

🧩 2. Flexbox — гибкая вёрстка

Flexbox — мощный инструмент для создания адаптивных и гибких макетов без костылей.

1️⃣ display: flex — включаем гибкость

Родитель становится flex-контейнером, дети — flex-элементами.

1
2
3

2️⃣ flex-direction — направление главной оси

row (строка) или column (столбец).

row 1
row 2
column 1
column 2

3️⃣ justify-content — выравнивание по главной оси

center — идеальный центр:

A
B

space-between — крайние по краям, остальные равномерно:

1
2
3

space-around — отступы вокруг каждого элемента:

X
Y
Z

4️⃣ align-items — выравнивание по поперечной оси

Элементы разной высоты — выравнивание по центру вертикали:

Малый
Средний
Большой

✅ align-items: center отлично центрирует по вертикали!

5️⃣ flex-wrap — перенос строк

Когда места мало — элементы переносятся на новую строку.

Карточка 1
Карточка 2
Карточка 3
Карточка 4
Карточка 5
Карточка 6

6️⃣ Гибкость элементов: flex-grow и order

flex-grow — коэффициент растяжения при свободном месте.

🌱 1 часть
🌿 2 части
📦 фикс

order — меняем порядок визуально (без изменения HTML).

🔹 Третий (order:3)
🔹 Первый (order:1)
🔹 Второй (order:2)

📌 Мой конспект по Flexbox

  • display: flex; — магия началась ✨
  • flex-direction — ось X (row) или Y (column)
  • justify-content — распределение по главной оси
  • align-items — по поперечной оси
  • flex-wrap: wrap — адаптив без боли
  • gap — воздух между элементами
  • flex-grow — кто больше хочет места
  • order — меняем порядок как захотим
